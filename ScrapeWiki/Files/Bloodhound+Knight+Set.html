            Bloodhound Knight Set is an Armor Set in Elden Ring. Bloodhound Knight Set, is a sturdy set made for the Bloodhound Knights for Damage Negation. The Bloodhound Knights are trained as hunters and known to be unshakable trackers. Sets of Armor in Elden Ring are composed of four armor pieces which are Helms, Chest Armor, Gauntlets, and Leg Armor.

             

            Where to find Bloodhound Knight Set in Elden Ring

            Drops in Gelmir Hero's Grave - Mt. Gelmir. [Map Link]

            The entire Bloodhound Knight Set drops in Gelmir Hero's Grave - Mt. Gelmir. Dropped by the Bloodhound Knight guarding the Gelmir Knight Set pickup. The set must then be picked up from the corpse of the Knight. See it on the Elden Ring Map here.

            To reach the knight, drop down to the rafters where the second chariot can be ridden, but instead jump to the window just besides, climb the ladder down and the Knight will be in front of you.

             

            Bloodhound Knight Set Armor Pieces in Elden Ring

            • Bloodhound Knight Helm
            • Bloodhound Knight Armor
            • Bloodhound Knight Gauntlets
            • Bloodhound Knight Greaves

             

            Bloodhound Knight Set Elden Ring Notes and Tips

            • As many of the comments mention, the helmet of this set is a bascinet with a "hounskull" visor attachment. As the name suggests, it is called this because it resembles the face of a hound. .
